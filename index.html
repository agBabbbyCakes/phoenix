Run Locally

Get Silverback running on your machine in minutes. No complex setup required.

Quick Start

Option 1: Using start.py (Easiest - Recommended)

# Clone the repository
git clone https://github.com/agBabbbyCakes/phoenix.git
cd phoenix
# Run the server
python start.py

This automatically handles Python version checks, dependency installation, and port selection. The server will start at http://localhost:8000

Option 2: Using uv (Fast & Modern)

# Install uv (if not installed)
curl -LsSf https://astral.sh/uv/install.sh | sh
# Run the server
uv run uvicorn app.main:app --reload --port 8000

Option 3: Using Python venv

# Create virtual environment
python3 -m venv .venv
# Activate it
source .venv/bin/activate # macOS/Linux
.venv\Scripts\activate # Windows
# Install dependencies
pip install -r requirements.txt
# Run the server
uvicorn app.main:app --reload --port 8000

Access the Dashboard

  • ✓ Main Dashboard: http://localhost:8000/
  • ✓ Logs Viewer: http://localhost:8000/logs
  • ✓ Daily Report: http://localhost:8000/report
  • ✓ Health Check: http://localhost:8000/health
