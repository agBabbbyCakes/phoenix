Overview

Silverback is a real-time dashboard for monitoring Ethereum bots, built with a lightweight, streaming-first stack.

Backend

  • • FastAPI — web framework and routing
  • • Uvicorn — ASGI server
  • • Jinja2 — server-rendered templates and partials
  • • SSE (sse-starlette) — server-sent events for live updates
  • • DataStore — in-memory aggregation of metrics/events
  • • Endpoints: JSON APIs for charts and bot health

Frontend

  • • Tailwind CSS (CDN) + DaisyUI — styling
  • • Alpine.js — lightweight UI state and interactivity
  • • HTMX + SSE — streaming HTML swaps for live metrics
  • • Chart.js (+ zoom, matrix, annotation plugins) — charts & heatmaps
  • • Charts.css — CSS-only tiny bar/sparkline visuals
  • • Three.js — 3D network/financial/point-cloud views

Templates & Pages

  • • IDE Dashboard (`ide-dashboard.html`) — default landing view
  • • TV Mode (`index.html`) — widescreen display view
  • • Explorer, Bots, Logs, Reports, Settings pages
  • • Global Sidebar (`partials/global-sidebar.html`) — unified nav

Live Data Flow

  • • Bot logs or mock publisher feed the DataStore
  • • SSE broker pushes partial HTML to the client
  • • Charts pull JSON from `/api/live/{metric}` when needed
  • • 3D canvases render via Three.js modules

Packaging & Desktop

  • • Briefcase config present — browser-based UI (native packaging optional)
  • • Dockerfile and Render configs for deployment
