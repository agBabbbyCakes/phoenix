Installation Instructions

macOS

  1. Download the DMG file (zip format)
  2. Extract the zip file to get the .dmg
  3. Open the DMG file
  4. Drag "Phoenix Dashboard.app" to your Applications folder
  5. Open the app from Applications

Windows

  1. Download the Windows installer (.exe)
  2. Run the installer and follow the setup wizard
  3. Launch "Phoenix Dashboard" from the Start menu

Python

  1. Download the wheel file (.whl)
  2. Install with: pip install phoenix_dashboard-1.0.0-py3-none-any.whl
  3. Run with: python -m phoenix_dashboard
